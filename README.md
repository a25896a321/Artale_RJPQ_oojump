## 🕹️ 操作流程與介面說明

### 1. 設定玩家
- 點擊色塊選擇**個人顏色**
- 輸入**暱稱**（1-10 位英數或中文）
- 點擊「✅ 確定暱稱」

### 2. 建立房間（房主）
1. 點擊「🏠 建立房間」
2. 填寫房間名稱（必填）、密碼（選填）
3. 勾選選項：
   - **自動標記可能性**（預設開啟）
   - **顯示成員名單**（預設開啟，房主可踢出成員）
4. 點擊「✨ 建立」→ 點擊「🔗 邀請連結」分享

### 3. 加入房間（兩種方式）
| 方式 | 步驟 |
|------|------|
| **房號加入** | 貼上房號（fullMyId），點擊「加入」 |
| **房名搜尋** | 輸入房間名稱 → 點擊「🔍 搜尋」→ 選擇結果加入 |

> 房名搜尋採用 SHA-256 雜湊將名稱對應至固定 PeerID，同名房間可直接被搜尋。

### 4. 地圖互動

| 操作 | 電腦 | 手機 |
|------|------|------|
| 標記正確（每層限 1 格） | 左鍵點擊 | 單點 |
| 標記死路（每層限 3 格） | 右鍵點擊 | 雙擊 |
| 取消自己的標記 | 再次點擊 | 再次點擊 |

### 5. 門格狀態說明

| 狀態 | 顏色 | 說明 |
|------|------|------|
| 正確標記 | 玩家個人色（有名字） | 該玩家的正確格 |
| 死路標記 | 無底色變化，僅顯示 `✗×N` | N 位玩家標記為死路 |
| 可能推算 | 🟣 紫框閃爍 + `?` + 玩家名 | 2+ 死路時系統推算可能玩家 |
| 唯一確定 | 🟡 金框閃爍 + `★` + 「僅剩此格」 | 此格必定是該玩家 |

### 6. 自動標記規則
- **2+ 玩家標記死路**：推算出剩餘可能玩家，顯示名字提示
- **3 玩家標記死路**：剩餘 1 名玩家確定，顯示「僅剩此格」金色閃爍
- **3 玩家已選正確路**：剩餘 1 名玩家確定，標記最後格
- **2 玩家已選正確路**：剩餘 2 格標記可能的 2 位玩家

### 7. 房間管理（房主）
- 「🔑 編輯密碼」：修改或移除密碼
- 「🗑️ 清空所有標記」：清空所有格（含顏色重置）
- 「🔨 重建房間」：重建房間
- 「🚪 退出 + 重新連線」：退出並回到首頁

### 8. 閒置保護
房間閒置超過 **1 小時**自動失效，提示重新創建。

## 📋 遊戲規則
- 每位玩家每層只能標記 **1 個正確格**（左鍵）
- 每位玩家每層最多標記 **3 個死路**（右鍵）
- 房間上限 **4 人**
- 各玩家可自訂**暱稱**與**顏色**
